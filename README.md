# 🚢 Titanic Data Cleaning & Visualization Project

## 📌 Overview

This project demonstrates Data Cleaning, Exploratory Data Analysis (EDA), and Data Visualization using the Titanic Dataset. The objective is to transform raw data into meaningful insights by handling missing values, removing duplicates, detecting outliers, and creating visual reports.

## 🎯 Objectives

- Clean and preprocess raw data
- Handle missing values and duplicates
- Detect and analyze outliers
- Perform exploratory data analysis
- Create meaningful visualizations
- Generate insights from data

## 🛠 Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Visualizations

- Survival Count Analysis
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Correlation Heatmap

## 🔍 Key Findings

1. Female passengers had a significantly higher survival rate.
2. First-class passengers were more likely to survive.
3. Most passengers were between 20 and 40 years old.
4. Fare prices were strongly related to passenger class.
5. Gender and class were major factors affecting survival.
